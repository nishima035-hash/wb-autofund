УСТАНОВКА WB AUTOFUND В CHROME

1. Распакуйте ZIP-архив в постоянную папку.
2. Откройте в Chrome: chrome://extensions
3. Включите «Режим разработчика» справа сверху.
4. Нажмите «Загрузить распакованное расширение».
5. Выберите распакованную папку chrome-extension.
6. Нажмите значок WB AutoFund и сохраните адрес панели на сервере.

Chrome не устанавливает обычный ZIP напрямую: его нужно распаковать, затем выбрать папку. Для публикации в Chrome Web Store потребуется отдельная регистрация разработчика и загрузка этого ZIP в кабинет магазина.
